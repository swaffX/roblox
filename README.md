# Neurovia AI Coder

🤖 Pure AI-Driven Coding Assistant for Roblox Studio

[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## ✨ Features

- **Multi-AI Support**: OpenAI GPT-4, Claude 3, Google Gemini
- **Natural Language**: No keywords required, just talk naturally
- **Auto-Apply Code**: Generated code appears instantly in your game
- **Context-Aware**: Analyzes your entire project structure
- **Modern UI**: Professional dark theme interface

---

## 🚀 Quick Start

### For New Contributors (Automated Setup)

```powershell
# One-click setup (installs everything)
quick-setup.bat

# Or step by step:
npm install
setup-watch.bat
npm run build
npm run install-plugin
```

### For Daily Development

```powershell
# Start watch mode (recommended - 35x faster!)
npm run watch

# Then in Roblox Studio:
# Plugins → Rojo → Connect to localhost:34872
```
