🤖 **Pure AI-Driven Coding Assistant** for Roblox Studio

## ✨ Features

### 🎯 Core Features
- **Pure AI-Driven**: No keyword detection, AI decides everything
- **Multi-AI Support**: OpenAI GPT-4, Claude 3, Google Gemini
- **Auto-Apply Code**: Any code generated by AI is automatically applied
- **Workspace Scanner**: Comprehensive project structure analysis
- **Intent Analyzer**: Understands what you want without keywords
- **Real-time Execution**: No templates, everything generated on-the-fly

### 🔥 What Makes It Special
- ❌ **No Keyword Requirements** - Say anything, AI understands
- ✅ **Automatic Code Application** - Code appears instantly in your game
- 🧠 **Context-Aware** - Knows your entire project structure
- 🚀 **Real-time** - No pre-made templates, pure AI generation
- 🎨 **Modern UI** - Dark theme, professional interface

## 🚀 Quick Start

### Installation

1. Clone the repository:
```powershell
git clone https://github.com/swaffX/neurovia-roblox.git
cd neurovia-roblox
```

2. Build the plugin:
```powershell
npm run build
```

3. Install to Roblox Studio:
```powershell
npm run install-plugin
```

### Usage

1. Open Roblox Studio
2. Click "Neurovia" in the Plugins tab
3. Select your AI provider (Gemini/Claude/OpenAI)
4. Enter your API key
5. **Just talk naturally!**

### Examples

`
You: "seviye sistemi"
AI: [Generates level system code]
Plugin: ✅ Auto-applied to your game!

You: "can you help me make a GUI"
AI: [Generates GUI code]
Plugin: ✅ Auto-applied!

You: "inventory system"
AI: [Generates inventory]
Plugin: ✅ Auto-applied!
`

**No keywords needed!** Just describe what you want.

## 🏗️ Architecture

### Core Modules
- **WorkspaceScanner**: Scans entire project structure
- **IntentAnalyzer**: Understands user intent without keywords
- **PromptBuilder**: Builds AI prompts with full context
- **ResponseParser**: Extracts and applies code automatically
- **APIManager**: Manages multi-AI provider communication

### Data Flow
`
User Input (ANY text)
    ↓
WorkspaceScanner (full project scan)
    ↓
IntentAnalyzer (what does user want?)
    ↓
AI Provider (Gemini/Claude/OpenAI)
    ↓
ResponseParser (extract code)
    ↓
Auto-Apply (no confirmation needed)
    ↓
✅ Code in your game!
`

## 🔧 Configuration

Edit src/Config.lua for customization:

```lua
Config.AI_PROVIDERS = {
    OPENAI = "OpenAI",
    CLAUDE = "Claude",
    GEMINI = "Gemini"
}

Config.DEFAULT_MODELS = {
    OPENAI = "gpt-4-turbo",
    CLAUDE = "claude-3-sonnet-20240229",
    GEMINI = "gemini-2.5-flash"
}
```
