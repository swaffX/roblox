# Neurovia Coder

🟥 AI-powered coding assistant for Roblox Studio

Multi-AI support: OpenAI GPT-4, Claude 3, Google Gemini

---

## 🚀 Quick Start

```powershell
git clone https://github.com/swaffX/neurovia-roblox.git rblx
cd rblx
```

Then ask **Warp AI**: **"Set up this project"**

Warp will automatically:
- Install dependencies
- Download Rojo plugin
- Build and install the plugin
- Start watch mode
- Guide you to connect Studio
